üìö Funci√≥n cv2.resize()

cv2.resize(imagen, (ancho, alto))

Redimensiona una imagen a las dimensiones especificadas. Es la forma m√°s directa de cambiar el tama√±o.

Sintaxis b√°sica: imagen_nueva = cv2.resize(imagen_original, (500, 400))

cv2.resize(imagen, None, fx=factor_x, fy=factor_y)

Redimensiona usando factores de escala en lugar de dimensiones absolutas. √ötil para mantener proporciones.

Ejemplo: fx=0.5, fy=0.5 ‚Üí Reduce la imagen al 50% de su tama√±o
fx=2.0, fy=2.0 ‚Üí Duplica el tama√±o de la imagen

Par√°metro: interpolation

Especifica el m√©todo de interpolaci√≥n usado para redimensionar. Afecta la calidad y velocidad del proceso.

M√©todos disponibles:
‚Ä¢ INTER_LINEAR: Interpolaci√≥n bilineal (por defecto, buena calidad)
‚Ä¢ INTER_NEAREST: Vecino m√°s cercano (m√°s r√°pido, menor calidad)
‚Ä¢ INTER_CUBIC: Interpolaci√≥n bic√∫bica (mejor calidad, m√°s lento)
‚Ä¢ INTER_AREA: Ideal para reducir tama√±o

‚öôÔ∏è M√©todos de Interpolaci√≥n Detallados

cv2.INTER_LINEAR

Interpolaci√≥n bilineal. Balancea velocidad y calidad. Es el m√©todo por defecto y funciona bien para la mayor√≠a de casos.

Cu√°ndo usar: Para redimensionamientos generales donde necesitas buen equilibrio entre velocidad y calidad.

cv2.INTER_NEAREST

Interpolaci√≥n por vecino m√°s cercano. El m√©todo m√°s r√°pido pero con menor calidad, puede producir im√°genes pixeladas.

Cu√°ndo usar: Cuando la velocidad es cr√≠tica o trabajas con im√°genes binarias/m√°scaras.

cv2.INTER_CUBIC

Interpolaci√≥n bic√∫bica sobre vecindario 4x4. Produce resultados m√°s suaves y de mejor calidad, pero es m√°s lento.

Cu√°ndo usar: Para agrandar im√°genes donde la calidad es prioritaria.

cv2.INTER_AREA

Remuestreo usando relaci√≥n de √°rea de p√≠xeles. M√©todo preferido para reducir el tama√±o de im√°genes.

Cu√°ndo usar: Espec√≠ficamente para reducir tama√±os, produce mejores resultados sin aliasing.

Recomendaciones de Rendimiento

Optimizaci√≥n:
‚Ä¢ Para agrandar: usa INTER_CUBIC o INTER_LINEAR
‚Ä¢ Para reducir: usa INTER_AREA
‚Ä¢ Para velocidad m√°xima: usa INTER_NEAREST
‚Ä¢ Para calidad m√°xima: usa INTER_CUBIC

üéØ Ejemplo Completo con Diferentes M√©todos

import cv2

# Cargar imagen
img = cv2.imread('imagen.jpg')
print(f"Tama√±o original: {img.shape}")

# Reducir con INTER_AREA (mejor para reducir)
img_small = cv2.resize(img, (200, 150), 
                       interpolation=cv2.INTER_AREA)
cv2.imwrite("reducida_area.jpg", img_small)

# Agrandar con INTER_CUBIC (mejor para agrandar)
img_large = cv2.resize(img, (1000, 750), 
                       interpolation=cv2.INTER_CUBIC)
cv2.imwrite("ampliada_cubic.jpg", img_large)

# Escalar al 50% con factores
img_half = cv2.resize(img, None, fx=0.5, fy=0.5,
                      interpolation=cv2.INTER_LINEAR)
cv2.imwrite("mitad.jpg", img_half)

print("Procesamiento completado!")
