Traducción de Texto con Python

Traducir texto automáticamente a múltiples idiomas usando deep_translator

📝 Código del Ejemplo

from deep_translator import GoogleTranslator


def translate_text(text, target_languages):
    translations = {}
    for lang in target_languages:
        try:
            translated = GoogleTranslator(source='es', target=lang).translate(text)
            translations[lang] = translated
        except Exception as e:
            translations[lang] = f"Error al traducir a {lang}: {e}"
    return translations


if __name__ == "__main__":
    input_text = input("Introduce el texto en español a traducir: ")

    # Idiomas objetivo (códigos ISO 639-1)
    # 'en' para inglés, 'fr' para francés, 'de' para alemán
    target_langs = ['en', 'fr', 'de']

    print(f"\nTexto original (español): {input_text}")

    results = translate_text(input_text, target_langs)

    for lang, translated_text in results.items():
        print(f"Traducción ({lang}): {translated_text}")

📚 Glosario de Funciones y Conceptos

deep_translator

Biblioteca de Python que proporciona una interfaz unificada para múltiples servicios de traducción automática. Permite traducir texto entre diferentes idiomas de manera sencilla y eficiente sin necesidad de configurar APIs complejas.

Instalación:

pip install deep-translator

GoogleTranslator

Clase de deep_translator que utiliza el servicio de Google Translate para realizar traducciones. Se inicializa especificando el idioma de origen (source) y el idioma de destino (target).

Parámetros:

source - Código del idioma de origen (ej: 'es' para español)

target - Código del idioma de destino (ej: 'en' para inglés)

translate()

Método que realiza la traducción del texto proporcionado. Recibe una cadena de texto en el idioma de origen y devuelve la traducción en el idioma de destino.

Códigos ISO 639-1

Estándar internacional de códigos de dos letras para representar idiomas. Son utilizados por los servicios de traducción para identificar los idiomas de manera uniforme.

Ejemplos comunes:

es - Español

en - Inglés

fr - Francés

de - Alemán

it - Italiano

pt - Portugués

ja - Japonés

zh - Chino

Funciones definidas (def)

Bloque de código reutilizable que realiza una tarea específica. En este caso, translate_text es una función que recibe un texto y una lista de idiomas, y devuelve un diccionario con las traducciones.

Parámetros de translate_text:

text - El texto a traducir

target_languages - Lista de códigos de idiomas destino

return - Diccionario con traducciones {idioma: texto_traducido}

try/except

Estructura de manejo de excepciones que permite capturar y gestionar errores durante la ejecución del código. Si ocurre un error al traducir a un idioma específico, el programa no se detiene, sino que registra el error y continúa con los demás idiomas.

Uso:

try: - Bloque de código que puede generar errores

except Exception as e: - Captura cualquier error y lo almacena en la variable e

items()

Método de diccionarios que devuelve una vista de pares clave-valor. Permite iterar simultáneamente sobre las claves (idiomas) y valores (traducciones) del diccionario.

Ejemplo:

for lang, translated_text in results.items():

Itera sobre cada idioma y su traducción correspondiente

if __name__ == "__main__":

Condicional especial que verifica si el script se está ejecutando directamente (no siendo importado como módulo). El código dentro de este bloque solo se ejecuta cuando el archivo se ejecuta directamente.

f-strings

Formato de cadenas en Python que permite insertar expresiones dentro de cadenas de texto usando llaves {}. Se identifican por la letra 'f' antes de las comillas.

Ejemplo:

f"Texto original (español): {input_text}"

Inserta el valor de input_text dentro de la cadena

Traducción automática

Proceso de convertir texto de un idioma a otro utilizando algoritmos de inteligencia artificial y procesamiento de lenguaje natural. Los servicios modernos como Google Translate utilizan redes neuronales para producir traducciones cada vez más precisas y naturales.
