üìö Propiedades de la Imagen

img.shape

Devuelve una tupla con las dimensiones de la imagen en el formato (alto, ancho, canales). Es la propiedad m√°s importante para conocer las caracter√≠sticas b√°sicas de una imagen.

Formato de salida: (altura_en_p√≠xeles, anchura_en_p√≠xeles, n√∫mero_de_canales)

img.shape[0] - Alto de la imagen

Representa la altura de la imagen en p√≠xeles, es decir, el n√∫mero de filas de p√≠xeles que contiene la imagen verticalmente.

Ejemplo: Si img.shape[0] = 480, significa que la imagen tiene 480 p√≠xeles de altura

img.shape[1] - Ancho de la imagen

Indica la anchura de la imagen en p√≠xeles, es decir, el n√∫mero de columnas de p√≠xeles que contiene la imagen horizontalmente.

Ejemplo: Si img.shape[1] = 640, significa que la imagen tiene 640 p√≠xeles de ancho

img.shape[2] - Canales de color

Especifica el n√∫mero de canales de color que tiene la imagen. Determina si es una imagen en escala de grises, color, o con transparencia.

Valores comunes:
‚Ä¢ 1 canal: Imagen en escala de grises
‚Ä¢ 3 canales: Imagen a color (BGR en OpenCV)
‚Ä¢ 4 canales: Imagen con transparencia (BGRA)

Ejemplo Completo de An√°lisis

import cv2
import os

def analizar_imagen(ruta_imagen):
    # Verificar si el archivo existe
    if not os.path.exists(ruta_imagen):
        print(f"Error: El archivo {ruta_imagen} no existe")
        return
    
    # Cargar la imagen
    img = cv2.imread(ruta_imagen)
    
    if img is None:
        print(f"Error: No se pudo cargar {ruta_imagen}")
        return
    
    # Informaci√≥n b√°sica
    print(f"=== An√°lisis de {ruta_imagen} ===")
    print(f"Dimensiones: {img.shape}")
    print(f"Alto: {img.shape[0]} p√≠xeles")
    print(f"Ancho: {img.shape[1]} p√≠xeles")
    print(f"Canales: {img.shape[2]}")
    print(f"Total p√≠xeles: {img.shape[0] * img.shape[1]:,}")
    print(f"Tama√±o en memoria: {img.size:,} elementos")
    print(f"Tipo de datos: {img.dtype}")
    
    # Tama√±o del archivo
    tama√±o_archivo = os.path.getsize(ruta_imagen)
    print(f"Tama√±o del archivo: {tama√±o_archivo:,} bytes")

# Usar la funci√≥n
analizar_imagen('gato.jpg')
